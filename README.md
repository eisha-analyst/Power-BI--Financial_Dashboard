# Power-BI--Financial_Performance_Dashboard
## Project Goal
Built an interactive Financial Dashboard in Power BI 
tracking $101M revenue across 10 countries (2022-2024)
with Budget vs Actual analysis.

## Tools Used
- Power BI Desktop
- DAX Measures
- Power Query
- CSV Data

## Results
- Revenue tracked across 10 countries
- Gross Profit & EBITDA analysis
- Budget vs Actual comparison
- Conditional formatting for variance
